## 🚀 Installing dependencies

To install, follow these steps:

With Yarn:

```
$ yarn install
```

With NPM:

```
$ npm install
```

## ☕ Using scripts


With Yarn:

```
$ yarn env-cmd -e local ts-node scripts/filename.ts
```
